# ConsoleCraft
 
A simple mnecraft clone running and rendering to a windows terminal.

## Controls

- esc - capture mouse
- WASD - to move
- Q - break block
- E - build block
- \- - lower render distance
- \+ - expand render distance
- num1 - color mode
- num2 - grayscale mode
- num4 - low color textures
- num5 - full textures
